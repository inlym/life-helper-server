# bruno 说明

## 介绍

本项目使用 [bruno](https://github.com/usebruno/bruno) 作为 API 调试工具，该工具的优点是：

1. 接口数据保存在本地，便于 git 管理。
2. 界面简单，无多余功能。

## 客户端下载
