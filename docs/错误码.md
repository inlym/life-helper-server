# 错误码

## 响应异常

当发生错误时，会由异常处理器捕获错误，并返回固定格式的响应。格式示例：

```json
{
  "errorCode": "1",
  "errorMessage": "网络异常，请稍后重试！"
}
```

## 错误码

### 全局错误码

全局错误码范围：1 - 99

| 错误码 | 响应状态码 | 说明                    |
|-----|-------|-----------------------|
| 1   | 500   | 兜底错误，用户处理未被任何处理器捕获的异常 |
| 2   | 401   | 需要登录的接口，用户未提供有效的鉴权信息  |
| 3   | 400   | 客户端参数错误               |
| 4   | 200   | 用户访问了不存在或者存在但不属于自己的资源 |

### 业务错误码

#### 第三方 API 模块

错误码范围：1001 - 1099

#### 用户模块

错误码范围：1101 - 1199

#### 登录模块

错误码范围：1201 - 1299
